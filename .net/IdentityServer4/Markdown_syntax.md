### Syntax

1. Headers

   ```markdown
   # h1 tag
   ## h2 tag
   ### h3 tag
   ...
   ```

   

2. Emphasis（强调）

   ```markdown
   *斜体*
   _斜体_
   **bold**
   __bold__
   __You **can** combine them__
   ```

3. Lists

   **无序**

   ```markdown
   * Item1
   * Item2
   	* Item3
   	* Item4	
   ```

   **有序**

   ```markdown
   1. Item1
   2. Item2
   	1. Item2.1
   ```

4. Images

   ```markdown
   ![name](path)
   ![logo](http://www.github.com/logo.png)
   ```

5. Links

   ```markdown
   automatic: http://github.com
   or
   [GitHub](http://github.com)
   ```

6. Blockquotes (引用)

   ```markdown
   > Markdown is best for all! --鲁迅	
   ```

   > Markdown is great!
